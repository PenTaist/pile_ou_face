FRENCH / FRANCAIS :

Voici le jeu du pile ou face !

Voici comment lancer le jeu :

1) téléchargez le repos
2) rendez vous dans le dossier ou se trouve le jeu
3) lancez un cmd
4) tapez "python pile_ou_face.py" ou "py pile_ou_face.py" cela dépend de la version de python
